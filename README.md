# PityRoll

An addon for WoW classic to implement a Pity Roll system for raiding.

## Installation

Move folder into AddOns directory. All user-specific data is stored in WTF.

## Usage

The addon provides the following slash commands:

- `/pityroll` or `/pr` - Show help message
- `/pityroll help` - Show all available commands
- `/pityroll version` - Display addon version
